## 🐛 常见问题排查

### 问题1: 页面空白
**检查**:
- 浏览器控制台是否有错误
- 确认npm run dev没有报错
- 清除浏览器缓存重试

### 问题2: 课程列表为空
**检查**:
- 后端服务是否启动
- Token是否有效
- Network面板查看API请求状态
- 后端日志是否有错误

### 问题3: 样式显示异常
**检查**:
- Tailwind CSS是否正确编译
- 浏览器是否缓存了旧样式
- globals.css是否正确导入

### 问题4: 路由跳转失败
**检查**:
- Next.js App Router配置
- 文件夹结构是否正确
- page.tsx文件是否存在

### 问题5: TypeScript错误
**检查**:
- 运行 `npm install` 确保依赖完整
- 检查 tsconfig.json 配置
- 重启VS Code TypeScript服务器
